Manifest Contract

Schema essenziale

{ "schema_version": "1.0", "pipeline": "smartbackup", "period": "2026-04", "period_type": "monthly", "status": "success", "started_at": "2026-05-01T10:00:00Z", "completed_at": "2026-05-01T10:08:43Z", "duration_seconds": 523, "clients_processed": ["KSGR", "DLPHRC"], "clients_failed": [], "reports": [ { "client": "KSGR", "type": "EXT", "s3_key": "smartbackup/reports/mensili/2026-04/SmartBackup_EXT_KSGR_2026-04.docx", "size_bytes": 47215 } ], "warnings": [], "errors": [] }

Implementazione nelle pipeline

from eko365_pipeline_core.runs import write_manifest, discover_reports # Alla fine del run, DOPO aws s3 sync report_entries = discover_reports( directory=Path("reports/output"), pipeline="smartbackup", period="2026-04", s3_prefix="smartbackup/reports", # ← NO period ) write_manifest( pipeline="smartbackup", period="2026-04", period_type="monthly", started_at=run_started_at, completed_at=datetime.now(timezone.utc), status="success", reports=report_entries, )
