Architettura Sorgente

La pipeline sfrutta un approccio email-based unico: Veeam Enterprise Manager (server on-prem EKOBKP08) invia report giornalieri via SMTP alla shared mailbox pipeline-ingest@eko365.it. L'app Entra ID EKO365-Pipeline-MailReader (client credentials, secret valido fino 02/2028) legge le email tramite Graph Mail API con Application Access Policy restrittiva.

Parsing HTML

Il parser gestisce sia formato nativo Veeam (hex color) che forward Outlook (rgb format). Estrae 9 colonne per task: resource_name, status, start/end_time, performance_rate, size, read, transferred, duration. Supporto formato europeo (virgola decimale: "1,1 TB").
