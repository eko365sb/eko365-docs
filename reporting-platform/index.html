Autenticazione

PRTG: API Key read-only come query parameter (apitoken=). Rate limit 5 req/min. Storico 365 giorni su Hosted. UniFi: API Key in header X-API-Key. EA endpoints, rate limit 100 req/min. ISP metrics 1h retention 30 giorni.

Note tecniche

Memory Active vs Consumed: i sensori VMware riportano RAM Consumed al 95-100% su Windows Server (cache filesystem). Il KPI corretto è Memory Active (working set reale). Storage: dati NetApp (fisico, post-dedup) solo nel bollettino interno. Il cliente paga il logico (IaaS). Fisico ×3 per stima costi EKO (dati + backup + DR).
